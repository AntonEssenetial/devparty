# DevParty Markup

Markup of the conference site.

## Usage

* Install [node.js](https://nodejs.org/en/)
* Install gulp `sudo npm install --global gulp-cli`
* Clone repo `git clone https://github.com/AntonEssential/devparty.git`
* `cd devparty`
* Run gulp `gulp`
